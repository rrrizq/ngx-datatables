ngx-datatables

An Angular library for building complex HTML tables using DataTables jQuery plug-in.

Features:
download
Quick Install
sync_alt
Angular integration
dns
Large dataset support
filter_alt
Advanced Data Filter
extension
Extensions support
lock_open
